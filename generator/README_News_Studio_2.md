# ZUSTAND News Studio 2.0

## Installation

Die drei Dateien aus diesem Paket in den vorhandenen Ordner

```text
z-panel/generator/
```

kopieren.

## Start

Doppelklick auf:

```text
news_studio_2_starten.bat
```

## Neue Funktionen

- übersichtliche Registerkarten
- Artikel und Entwürfe anzeigen
- neue Artikel über ein Formular erstellen
- vorhandene Artikel bearbeiten
- Artikel als Entwurf speichern
- Artikel freigeben und automatisch nach `newsredaktion/artikel/` verschieben
- Quellen über ein Formular erstellen und bearbeiten
- Bild- und Metadatendateien auflisten
- vollständige Prüfung und Erzeugung von `news.json`
- Projektübersicht mit aktuellen Beständen

## Hinweis zum Artikeltext

Version 2.0 speichert den Artikeltext zunächst als einen zusammenhängenden Textabschnitt. Strukturierte Zwischenüberschriften, Faktenblöcke und Linklisten können in einer späteren Ausbaustufe ergänzt werden.
