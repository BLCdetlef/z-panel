# ZUSTAND News Studio 3.0a

## Installation

Die beiden Programmdateien

- `news_studio_3a.py`
- `news_studio_3a_starten.bat`

in den vorhandenen Ordner

```text
z-panel/generator/
```

kopieren.

Die bisherige Version 2.0 kann zunächst im Ordner bleiben. Version 3.0a wird mit

```text
news_studio_3a_starten.bat
```

gestartet.

## Verbesserungen gegenüber 2.0

- scrollbarer Artikel-Editor
- feste Schaltflächenleiste am unteren Rand
- kleinere Mindestfenstergröße für Notebook-Bildschirme
- dauerhafte Statusleiste am unteren Fensterrand
- deutlichere Erfolgsmeldungen
- Artikelliste mit sichtbaren Spalten für Status, ID und Titel
- Statusänderungen nach dem Freigeben sind in der Liste erkennbar
- Doppelklick auf einen Artikel öffnet ihn
- Generator meldet die Zahl der erzeugten Artikel deutlich

## Hinweis

Version 3.0a verändert die vorhandenen Datenordner und JSON-Dateien nicht. Sie greift auf dieselben Ordner wie Version 2.0 zu:

```text
newsredaktion/entwuerfe/
newsredaktion/artikel/
newsredaktion/quellen/
assets/images/
```
